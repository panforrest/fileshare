# fileshare
File sharing app with realtime firebase apps

Setup Instructions:

1. clone
2. $ touch .env
3. In env file, add 'MONGO_URL=mongodb://localhost/fileshare'
4. $ npm install
5. In separate terminal tab, $ mongod
6. $ nodemon OR $ npm start
7. In browser, navigte to http://localhost:3000
8. PROFIT!
